Competition-based Neural Network

This project is a general case implementation of a local winner-take-all (LWTA) artificial neural networks, which is a special case of the more general competition-based networks. Inspired by recent brain studies, competition-based networks use the competition between neighboring neurons as a form of nonlinearity that enables the network to learn complex discriminant functions. Results show that these networks can achieve comparable accuracies to other state-of-the-art methods such rectified linear activation (ReLU) methods. LWTA neural networks, however, are less susceptible to the problem of catastrophic forgetting.

Motivation Studies of biological neural circuits such as [1] show a recurrent on-center, off-center anatomy of neurons that can be iterpretted as local competition between neighboring neurons. This behavior has recently inspired the design of artificial neural networks. It is therefore interesting to study what the advantages of such netowrks are compared to other types of neural networks.

Approach This work is largely inspired by the ideas in [2]. In this approach, each layer consists of blocks of neurons. For each specific input, only one linear neuron in each of these blocks is activated: the neuron with the highest rate of firing.

Code The code is open-source and is hosted on github.

Tools We implemented this project in MATLAB and used MNIST dataset to evaluate the model.


  • [1] Stephen Grossberg. Contour enhancement, short-term memory, and constancies in reverberating neural networks. Studies in Applied Mathematics, 52:213–257, 1973.
  • [2] Srivastava, Rupesh K., Masci, Jonathan, Kazerounian, Sohrob, Gomez, Faustino, and Schmidhuber, Jurgen. Compete to compute. In Advances in Neural Information Processing Systems, pp. 2310–2318, 2013.
