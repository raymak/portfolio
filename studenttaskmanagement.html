Task Management Tool

This project aims to integrate calendar and todo lists in a single tool.

Motivation Although there are many well-designed calendar and todo list applications, not many of them integrate these tworelated but conceptually different tools together. This integration is especially intended to be used by students due to their dense schedules with many due dates.

Prototype A med-fi prototype of this project is hosted here.

Tools We used Balsamique and paper for low-fi prototyping and HTML/Javascript for med-fi prototyping.
